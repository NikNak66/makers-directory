This digital download provides all the instructions on how to create a comfy, colourful and warm handmade cardigan to wear everywhere.

This pattern is suitable for beginners as it has easy to follow instructions with simple stitches as well as explanations on how to use different colours.

Other aspects to help make the project simple are detailed pictures to help with purchasing the correct yarn.

The pattern has a US-UK conversion table.

How was it made

Before Desi can create her DIY Crochet Patterns she had to first crochet the very first version. Whilst crocheting Desi will make notes of the steps whilst also taking photos.

Once this has done she is then able to create a first draft of the pattern. Before selling the pattern Desi will ask friends family or customers to test the instructions.
